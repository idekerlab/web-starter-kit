What is CyGraphShare?

CyGraphShare is an easy way to share your graph data files created with Cytoscape.

Easy to Use
Users can share Cytoscape.js interactive visualizations without programming.
Simple
Once file is available as a public URL, it can be shared by copy-paste it
Support Desktops and Tablets
It is a pure HTML5 application and supports both desktops and tablets.
Open Source
CyGraphShare is an open source project. Developers can use it as a boilarplate code for their web applications.

This is a pirot project towards the Cytoscape Cyberinfrastructure.
